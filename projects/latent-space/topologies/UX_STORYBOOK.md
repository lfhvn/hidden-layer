# UX Storyboard — Latent Topologies (mobile)

## Screen 1: Splash / Map Load
- Animated constellations, "Touch to enter"
- Progress bar for model init

## Screen 2: Explorer Map
- Pinch zoom / drag pan
- Points glow by density
- Tap → drawer opens (text, neighbors)
- Pitch & haptic feedback on movement

## Screen 3: Interpolation View
- Two selected points → scrubber
- Generated blend text updates live
- Audio glides in pitch

## Screen 4: Creator Mode
- Text input → local embed → placement animation
- Option to tag and save

## Screen 5: Notes / Journal
- Scrollable timeline of annotations
- Export JSON or video of latent walk

## Micro-interactions
- Crossing cluster → short vibrate
- Cluster core → lower pitch
- Long-press shows cosine neighbors
