# User Study Protocol — Latent Topologies

### Purpose
Investigate how humans perceive semantic relations while navigating latent space.

### Participants
10–15 designers / philosophically inclined users (18 +).

### Procedure
1. Intro + training (5 min)
2. Free exploration (10 min) — record screen + movement logs.
3. Guided tasks (e.g., find opposites, find analogies).
4. Post-experience interview (10 min).

### Data Collected
- Paths (x,y,t)
- Selected pairs / notes
- Transcribed verbal protocols

### Analysis
- Compare user-perceived clusters vs embedding neighbors.
- Thematic analysis of metaphors & emotions.

### Ethics
- Anonymous data; local storage only; consent recorded in-app.
